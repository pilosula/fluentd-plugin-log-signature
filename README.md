## Installation

### RubyGems

```
$ gem install fluent-plugin-log-signature
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-log-signature"
```

And then execute:

```
$ bundle
```

## Configuration

You can generate configuration template:

```
$ fluent-plugin-config-format filter log-signature
```
